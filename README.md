# 42Seoul-get_next_line

## 목표
- 주어진 fd가 가리키는 파일의 내용을 한줄 씩 반환하는 함수 구현.
- 더 이상 읽을 것이 없거나, 오류 발생시 NULL 반환.

## 관련 개념
- 정적 변수(Static variable).
- 함수 매개 변수로 이중 포인터 사용.

## 개발 및 테스트 환경
- Intel CPU 기반 Mac.
- MacOS Version = 12.6.2

## 사용 방법
- libft.a 라이브러리 필요.
- libft.h, get_next_line.h 헤더 파일 include 후, get_next_line 함수 사용.
